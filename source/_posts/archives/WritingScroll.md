## 文章发布基本流程

### 新建文件

```bash
hexo new butt {{title}} -p {{folder}}/{{file_name}}
```

* `{{title}}`标题
* `{{folder}}`文件夹
* `{{file_name}}`文件名

例如

```bash
hexo new butt "写作宝典" -p archives/WritingScroll
```
